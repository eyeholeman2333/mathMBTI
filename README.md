# MITI — 数学直觉类型测试

**Mathematical Intuition Type Indicator**

> 通过 16 道情境题，揭示你在数学世界中的认知偏好与思维风格。

---

## ✨ 简介

MITI 是一个受 MBTI 启发、专为数学/理工科人设计的**趣味性格测试**。它从四个维度描述你的数学直觉类型，共可产生 **16 种数学人格**。

灵感来源：与 Qwen3-Max 的一次关于"信息论 vs 泛函分析"思维风格差异的对话。

---

## 🧠 四个维度

| 维度 | 极性 A | 极性 B | 描述 |
|------|--------|--------|------|
| **对象偏好** | D 离散 | C 连续 | 偏好处理离散结构（图、集合）还是连续对象（函数、流形）？ |
| **推理风格** | C 构造 | E 存在 | 关注"如何构造出解"还是"解是否存在/唯一"？ |
| **信息态度** | P 概率 | D 决定 | 接受不确定性用概率建模，还是追求确定性与精确性？ |
| **结构视角** | G 全局 | L 局部 | 先看整体结构与对称性，还是从局部细节入手？ |

---

## 🔢 部分类型示例

| 类型 | 名称 | 适合方向 |
|------|------|----------|
| DCPG | 信息架构师 | 生成模型、信息论、图神经网络 |
| CEEL | 分析哲人 | 泛函分析、PDE、微分几何 |
| DCDtG | 组合决策家 | 算法设计、组合优化、复杂度理论 |
| CCPG | 随机分析师 | 随机分析、金融数学、扩散模型 |
| CEEG | 拓扑哲学家 | 代数拓扑、范畴论、同调代数 |

## 🚀 快速开始

本项目为纯静态网页，**无需任何构建工具或服务器**：

```bash
# 克隆仓库
git clone https://github.com/your-username/miti-test.git
cd miti-test

# 直接用浏览器打开
open index.html   # macOS
start index.html  # Windows
```

或者直接将仓库部署到 **GitHub Pages**：

1. 进入仓库 → Settings → Pages
2. Source 选择 `main` 分支 / `/ (root)`
3. 保存后访问 `https://your-username.github.io/miti-test/`
